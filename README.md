### Tecnologias utilizadas

- React
- TypeScript
- Styled-components
- Material UI
- Jest

### Pré requisitos

- yarn
- eslint
- node
- typescript

### Plugins para VS Code

- ESLint
- Prettier
- EditorConfig

#### Instalações da aplicação

```
yarn install
```

#### Ambimente de desenvolvimento

```
yarn start
```

Disponível em http://localhost:3000

#### Executar os testes

```
yarn jest
```
